# Guía de demo comercial

## Objetivo

Mostrar en 10 a 15 minutos que el sistema resuelve la operación diaria de una veterinaria.

## Guion sugerido

1. Dashboard: mostrar resumen general.
2. Clientes y pacientes: crear un cliente y asociarle una mascota.
3. Agenda: cargar un turno.
4. Clínica: registrar atención, vacuna y receta.
5. Documentos: imprimir receta o historia clínica profesional.
6. Venta: vender producto o servicio.
7. Caja: mostrar movimiento generado.
8. Cuenta corriente: explicar pagos pendientes.
9. Stock: mostrar descuento y movimientos.
10. Cierre de caja: cerrar el día.
11. Reportes: filtrar por fecha.
12. Usuarios: mostrar permisos por rol.
13. Auditoría: mostrar trazabilidad.
14. Respaldo: exportar backup JSON.
15. Producción: mostrar checklist técnico.

## Mensaje comercial

El sistema no es solo una agenda: integra clínica, caja, stock, documentos, reportes, roles y auditoría.
